# CodeRoot Hub 🚀

Welcome to **CodeRoot Hub**, an educational platform designed for learning web development and programming. This project was developed as part of the NTI (National Telecommunication Institute) Aswan "Web Designer" training program.

## 📝 Description
CodeRoot Hub is a bilingual (Arabic-focused) web application that provides resources, courses, and information for aspiring developers. It simplifies the learning path for various technologies including HTML, CSS, JavaScript, Python, and more.

## ✨ Features
- **Comprehensive Courses**: Structured lessons on web technologies.
- **Responsive Design**: Optimized for various screen sizes using Bootstrap and custom CSS.
- **Interactive UI**: User-friendly navigation and engaging elements.
- **Bilingual Support**: Primarily in Arabic for better accessibility for local learners.

## 🏗️ Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Libraries/Frameworks**: Bootstrap 5, jQuery, Font Awesome
- **Fonts**: Google Fonts (Open Sans), Material Icons

## 📂 Project Structure
- `/assets`: Contains images, styles, and scripts.
- `/pages`: Additional site pages like About Us, Contact, and Auth.
- `/pages/courses`: Specific course modules for different programming languages.

## 🚀 Getting Started
1. Clone the repository.
2. Open `index.html` in your favorite browser.
3. Explore the various courses and sections!
